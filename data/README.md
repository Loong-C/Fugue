# Data

Use `scripts/fetch_resources.py` to populate this directory.

Suggested layout:

- `data/raw/humdrum/bach-wtc-fugues/`: voice-separated Well-Tempered Clavier fugues.
- `data/raw/humdrum/bach-wtc/`: full Well-Tempered Clavier Humdrum edition.
- `data/raw/jsb-chorales-dataset/`: JSB chorales in fixed-grid formats.
- `data/processed/`: normalized training/evaluation data produced by future preprocessing.

Raw datasets are ignored by git. Check each source license before redistributing any corpus.
